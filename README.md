🌤️ Glassmorphism Weather Forecast Web App

A beautiful, lightweight, single-page weather application built using raw **HTML**, dynamic asynchronous **JavaScript**, and **Tailwind CSS**. It delivers real-time weather information with a modern, glassmorphism-styled UI container.

## 🚀 Features

* **Real-Time Data:** Seamlessly integrates with the WeatherAPI to fetch accurate global weather statistics.
* **Modern UI/UX:** Styled using fluid glassmorphism aesthetics, Google fonts (`Plus Jakarta Sans`), and animated responsive elements.
* **Smart Interaction:** Supports continuous workflow diagnostics like input submission via the `Enter` key, immediate error validation alerts, and responsive loading indicators.
* **Zero Overhead:** A standalone frontend ecosystem with no build steps required—just open the file and go.
